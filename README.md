# LifeTether Release

Repository officiel de distribution de **LifeTether**.

Ce dépôt contient uniquement les **versions compilées** de l'application.  

## 📱 À propos de LifeTether

**LifeTether** est une application d'urgence conçue pour fournir rapidement des informations vitales et des outils utiles en situation critique.

Fonctionnalités principales :

- 📞 Liste des numéros d'urgence par pays
- ❤️ Carte des défibrillateurs et hôpitaux
- 🚑 Parcours d'urgence interactif pour guider les premiers secours
- 🎒 Aide à la préparation d'un kit d'urgence 72h
- ⚡ Application rapide, sans publicité et sans création de compte

L'objectif de LifeTether est simple : **rendre les informations de secours accessibles immédiatement**.

## ⚠️ Avertissement

LifeTether est une application d'assistance.  
Elle ne remplace **en aucun cas les services d'urgence officiels**.

En cas de situation critique, contactez immédiatement les services de secours de votre pays.
